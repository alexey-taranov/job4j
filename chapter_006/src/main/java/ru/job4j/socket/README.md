# Бот

Бот - мудрый Оракл. Реализация серверной и клиентской стороны.
Пользователь вводит данные в консоли и получает ответ с сервера.
Оракл отвечает на приветствие Hello Oracle. Если Ораклу сказали Exit, то приложение выключается.
На любую другую слово-фразу, сервер отвечает I don't understand you.
Оракл может отправлять большие сообщения. Чтобы понять, когда конец сообщения, он отправляет пустую строку.
